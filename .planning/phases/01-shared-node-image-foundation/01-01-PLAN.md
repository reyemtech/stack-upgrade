**detect-stack.sh** (`stacks/node/scripts/detect-stack.sh`):
Standalone script that outputs a single word to stdout: `nextjs`, `cra`, or `vite-react`. Exits 1 with clear error on no match.

Implementation (per research Pattern 1):
- Accept optional arg for repo dir, default `/workspace`
- Check `package.json` exists, exit 1 if missing
- `has_dep()` helper using `jq -e` to check both `dependencies` and `devDependencies`
- Priority 1: `has_dep "next"` OR `ls "$REPO_DIR"/next.config.* 2>/dev/null` -> echo "nextjs"
- Priority 2: `has_dep "react-scripts"` -> echo "cra"
- Priority 3: `has_dep "@vitejs/plugin-react"` OR `has_dep "@vitejs/plugin-react-swc"` -> echo "vite-react"
- No match: print error to stderr listing found deps and supported stacks, exit 1

Use `set -e` at top. Add `#!/bin/bash` shebang.

6. **Stack detection** (NEW — not in Laravel):
   ```bash
   AUTO_DETECTED=$(/skill/scripts/detect-stack.sh /workspace)
   if [ -n "$STACK_TYPE" ]; then
     if [ "$STACK_TYPE" != "$AUTO_DETECTED" ]; then
       echo "WARNING: STACK_TYPE='$STACK_TYPE' but auto-detection found '$AUTO_DETECTED'. Using override."
     fi
   else
     STACK_TYPE="$AUTO_DETECTED"
   fi
   export STACK_TYPE
   echo "Stack type: $STACK_TYPE"
   ```

7. **Package manager detection** (NEW):
   ```bash
   if [ -f pnpm-lock.yaml ]; then PKG_MANAGER="pnpm"
   elif [ -f yarn.lock ]; then PKG_MANAGER="yarn"
   else PKG_MANAGER="npm"; fi
   export PKG_MANAGER
   echo "Package manager: $PKG_MANAGER"
   ```

8. **Dependency install** (replaces composer install):
   Best-effort, non-fatal. Use detected PKG_MANAGER:
   - pnpm: `pnpm install --frozen-lockfile 2>/dev/null || pnpm install`
   - yarn: `yarn install --frozen-lockfile 2>/dev/null || yarn install`
   - npm: `npm ci 2>/dev/null || npm install`
   Wrap in `|| echo "WARNING: Dependency install failed. Claude Code will handle this."`

9. **Before-snapshots** (INFRA-09):
   ```bash
   jq '{dependencies: (.dependencies // {}), devDependencies: (.devDependencies // {})}' package.json > /output/before-package-deps.json 2>/dev/null || echo "{}" > /output/before-package-deps.json
   for lockfile in package-lock.json yarn.lock pnpm-lock.yaml; do
     [ -f "$lockfile" ] && sha256sum "$lockfile" > /output/before-lockfile-hash.txt && break
   done
   ```

10. Baseline verification: `/skill/scripts/verify-full.sh 2>&1 | tee /output/baseline.log || echo "WARNING: Baseline verification had failures (expected pre-upgrade)."`
