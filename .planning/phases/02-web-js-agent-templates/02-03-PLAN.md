Replace both placeholder files. Both go through envsubst — use `${TARGET_VITE}`, `${TARGET_REACT}`, `${UPGRADE_DATE}`, `${REPO_URL}`.

**plan.md** — Header: "Vite + React Upgrade Plan", Target Vite: ${TARGET_VITE}, Target React: ${TARGET_REACT}, Date: ${UPGRADE_DATE}, Repo: ${REPO_URL}. Then 5 sequential phases:

Phase 1: React Version Bump + Codemod (VITE-01)
- `npm install react@${TARGET_REACT} react-dom@${TARGET_REACT}` (or pnpm/yarn equivalent based on package manager)
- Also update `@types/react` and `@types/react-dom` if TypeScript project
- Run `npx react-codemod` for breaking change transforms (forwardRef removal, useFormState→useActionState, act import paths)
- Commit codemod output separately
- Verify: `.upgrade/scripts/verify-fast.sh`

Phase 2: Vite Version Bump + Plugin Update (VITE-02)
- `npm install vite@${TARGET_VITE} @vitejs/plugin-react@latest`
- Update any other Vite plugins to compatible versions
- Verify: `.upgrade/scripts/verify-fast.sh`

Phase 3: Vite Config Format Changes (VITE-03)
- Check vite.config for deprecated/changed options:
  - Remove `legacy.proxySsrExternalModules` if present (Vite 6)
  - If `resolve.conditions` is set with custom values, add `...defaultClientConditions` or `...defaultServerConditions` imports from 'vite'
  - If `css.preprocessorOptions.sass.api: 'legacy'` exists, remove it (Vite 6 requires modern Sass API)
  - Check `json.stringify` interaction with `json.namedExports` (Vite 6 change)
  - If using postcss-load-config with TypeScript, verify compatibility
- Verify: `vite build` passes

Phase 4: TypeScript Types Codemod (VITE-05)
- Check if `tsconfig.json` exists — skip this phase entirely if not a TypeScript project
- Run `npx types-react-codemod@latest preset-19 ./src`
- This handles all @types/react breaking type changes for v19
- Commit codemod output
- Verify: `tsc --noEmit` passes (or skip if no tsconfig)

Phase 5: Build Verify + Dependency Cleanup (VITE-04)
- Run `.upgrade/scripts/verify-full.sh` — full build + lint + tests + npm audit
- Run `npm outdated` and update remaining outdated packages
- Remove unused packages
- Final `.upgrade/scripts/verify-full.sh`
- Verify: verify-full.sh passes clean

End with Constraints section.
